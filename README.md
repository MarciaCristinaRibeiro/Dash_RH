# Dash_RH
**Exercício 1 - Dashboard de RH no Google Sheets**

### 🎯 <ins> Objetivo </ins>

   * Criar um dashboard de RH no Google Sheets.

### 📄 <ins> Sobre a Base de Dados </ins>

   * A base de dados apresentada é fictícia de 2013 a 2018, criada no google sheets referente ao exercício prático do Bootcamp de Analista de Dados da SoulCode Academy.

### 📝 <ins> Perguntas de Negócios: </ins>

  1. Quantas contratações a empresa possui?
  2. Dessas contratações quantos colaboradores estão ativos e desligados, separados por gênero, a empresa emprega atualmente?
  3. Mostrar a quantidade de colaboradores ativos e desligados de 2013 a 2018.
  4. Qual o tempo de empresa dos colaboradores?
  5. Mostrar a quantidade de colaboradores por área de negócio e nível de contratação.
  6. Trazer um resumo de salário.
  7. Demonstrar o desempenho por colaborador.
